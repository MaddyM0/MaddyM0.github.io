How to use the PokéDirectory

Welcome!

The PokéDirectory is a web application where a User can enter in a Pokémons name in the search bar and recieve statistics about selected pokémon on the webpage (Photo of Pokémon, Name, Abilities, Height, Weight).

Users can select from the first 1,050 Pokemon.

Click here for Pokemon List

*At the moment this application is case sensitive.*
